# 🚦 Smart Traffic Violation Pattern Detector

An intelligent, data-driven web application that analyzes traffic violation data to identify patterns, hotspots, trends, and predict future traffic violations. The system provides interactive dashboards, visualizations, reports, and machine learning-based predictions to support smarter traffic monitoring and decision-making.

## 📖 Overview

The Smart Traffic Violation Pattern Detector helps users analyze historical traffic violation datasets through an interactive dashboard. It identifies violation hotspots, visualizes trends, predicts future violations using machine learning, and generates downloadable reports.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/devivittanala/Smart-Traffic-Violation-Pattern-Detector.git
```

### Navigate to the Project

```bash
cd Smart-Traffic-Violation-Pattern-Detector
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
